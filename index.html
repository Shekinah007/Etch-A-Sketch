Etch-A-Sketch

    Instructions

  • Click once with mouse over canvas to activate/deactivate cursor.
  • Alternatively, hold shift over canvas to activate the cursor.
  • The brush icon indicates when the cursor is active.
  • You can change the color of the canvas and brush using the color pickers.
  • Toggle the eraser tool by pressing "E" or clicking on the eraser icon.
  • Toggle view of the grid lines using the "Show Grid" button.
  • Use the slider to change grid size.

      • Note: Changing the grid size will refresh the canvas.
